# Руководство по размещению сайта в интернете

## Варианты размещения

### 1. GitHub Pages (Бесплатно, рекомендуется)

**Шаги:**
1. Создайте аккаунт на [GitHub.com](https://github.com)
2. Создайте новый репозиторий (например, `diamonds-website`)
3. Загрузите все файлы (index.html, styles.css, script.js) в репозиторий
4. Перейдите в Settings → Pages
5. Выберите ветку `main` и папку `/ (root)`
6. Сохраните - ваш сайт будет доступен по адресу: `https://ваш-username.github.io/diamonds-website`

**Команды для загрузки через Git:**
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ваш-username/diamonds-website.git
git push -u origin main
```

### 2. Netlify (Бесплатно, очень просто)

**Шаги:**
1. Зайдите на [Netlify.com](https://www.netlify.com)
2. Зарегистрируйтесь (можно через GitHub)
3. Перетащите папку с файлами в окно браузера
4. Сайт будет автоматически размещен и получит адрес типа `diamonds-juwelier.netlify.app`
5. Можно настроить свой домен (например, diamonds-juwelier.pro)

### 3. Vercel (Бесплатно)

**Шаги:**
1. Зайдите на [Vercel.com](https://vercel.com)
2. Зарегистрируйтесь
3. Импортируйте проект (можно через GitHub или загрузить файлы)
4. Сайт будет размещен автоматически

### 4. Cloudflare Pages (Бесплатно)

**Шаги:**
1. Зайдите на [Cloudflare.com](https://www.cloudflare.com)
2. Создайте аккаунт
3. Перейдите в Pages → Create a project
4. Загрузите файлы или подключите GitHub репозиторий

### 5. Платный хостинг (для домена .pro)

Если у вас уже есть домен `diamonds-juwelier.pro`:

**Популярные хостинги:**
- **Hostinger** - от €2-3/месяц
- **Namecheap** - от $3/месяц
- **IONOS** - от €1/месяц (хорошо для Германии)
- **Strato** - популярен в Германии

**Шаги:**
1. Купите хостинг
2. Загрузите файлы через FTP или файловый менеджер
3. Настройте DNS для домена

## Рекомендация

Для начала используйте **Netlify** или **GitHub Pages** - это бесплатно и очень просто. Позже можно подключить свой домен.

## Настройка домена

Если хотите использовать домен `diamonds-juwelier.pro`:

1. Купите домен (например, на Namecheap, GoDaddy, или IONOS)
2. В настройках Netlify/GitHub Pages добавьте свой домен
3. Настройте DNS записи согласно инструкциям хостинга
